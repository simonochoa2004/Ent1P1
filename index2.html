CONTAR


Esta función permite contar el numero de datos seleccionados en un rango en este caso los datos de Precio.












CONTAR.SI


Esta función permite contar el número de celdas seleccionados en un rango que cumplen con cierta condición, en este caso los dstos de Precio mayores a 200.












CONTAR.SI.CONJUNTO


Esta función permite contar el número de datos de un rango condicionado por los datos de otro, en este caso los productos con nombre "Sal" cuya cantidad sea menor a 20.
